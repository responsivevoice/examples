npm + Vite Example

Demo Mode

ResponsiveVoice imported from npm as an ESM module and bundled by Vite.

How to run this example
  1. Get a free API key at responsivevoice.org/register
  2. Install deps: npm install
  3. Start the Vite dev server: npm run vite:dev
  4. Edit src/main.ts — uncomment the apiKey field in rv.init({ ... }) and paste your key
  5. Open http://localhost:5173/

For a production bundle: npm run vite:build && npm run vite:preview. If you're reading this page from a plain static server (not Vite), the <script> tag below will 404 — that's expected; Vite's dev server transpiles the TypeScript entry on the fly.

Same UI as the basic example, but it imports @responsivevoice/core from npm via import { getResponsiveVoice } from '@responsivevoice/core' in src/main.ts — no CDN script tag, no IIFE global. Vite bundles the TypeScript entry for the browser.
